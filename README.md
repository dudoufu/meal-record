# 🍽️ 干饭日记 — 微信小程序

> 极简饮食记录小程序 · 10秒记录一餐

## 📱 功能概览

| 页面 | 功能 |
|------|------|
| 🏠 **首页** | 时段问候、今日记录数、连续天数🔥、好吃榜、最近记录 |
| 📸 **记录饮食** | 拍照/相册上传 → AI识别食物 → 编辑表单 → 保存 |
| 📊 **记录浏览** | 日期条日历、按餐次分组、本周好吃分布、分类饼图、排行榜 |
| 👤 **我的** | 个人信息、照片墙、数据报告（预留） |
| 📋 **记录详情** | 查看详情、修改好吃程度、删除记录 |

## 🚀 快速开始

### 1. 下载微信开发者工具

前往 [微信开发者工具下载页](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载并安装。

> 选择 **稳定版 (Stable Build)** 即可

### 2. 打开项目

1. 打开微信开发者工具
2. 点击 **「小程序」** → **「导入项目」**
3. 项目目录选择：`d:\work\workfile\干饭日记`
4. AppID：选择 **「测试号」**（无需注册，直接使用）
5. 点击 **「导入」**

### 3. 预览

导入后即可在模拟器中预览。支持：

- **真机预览**：点击工具栏「预览」→ 用微信扫码
- **编译模式**：可在「编译」下拉选择指定页面

## 🔧 开发说明

### 数据存储

所有记录存储在微信本地缓存，key 为 `food_records`。启动时自动初始化8条种子数据。

```javascript
// 记录数据结构
{
  id: "自动生成",
  foodName: "红烧牛肉面",
  category: "主食",           // 主食/肉类/蔬菜/水果/饮料/其他
  mealType: "lunch",         // breakfast/lunch/dinner/snack
  tastiness: "amazing",       // amazing/good/okay/bad
  notes: "汤很好喝",
  imageUrl: "",               // 云存储地址（预留）
  createdAt: 时间戳,
  updatedAt: 时间戳
}
```

### 好吃程度体系

| 等级 | 标识 | 表情 | 颜色 |
|------|------|------|------|
| 超赞 | amazing | 😋 | 金色 #f59e0b |
| 好吃 | good | 🤩 | 绿色 #10b981 |
| 一般 | okay | 😐 | 蓝色 #6366f1 |
| 踩雷 | bad | 😣 | 红色 #ef4444 |

### AI识别

当前为模拟识别（随机从内置食物库选取），接入真实AI服务：

1. 在 `pages/record/record.js` 的 `simulateRecognition` 方法中替换调用
2. 可接入：腾讯云慧眼 / Claude视觉 / GPT-4o
